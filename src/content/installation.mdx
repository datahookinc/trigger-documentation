# Installation

**Trigger** does not have any dependencies, so installation into your project is simple and lightweight:

```bash
npm i @datahook/trigger
```

Trigger uses *semver* naming conventions, so you can install a specific version if needed:

```bash
npm i @datahook/trigger@0.20
```

You can review the entire [npmjs version history](https://www.npmjs.com/package/@datahook/trigger?activeTab=versions) for **Trigger**

<mark>**Note:** It is recommended that you use **Trigger** with **TypeScript** for an optimal developer experience</mark>
